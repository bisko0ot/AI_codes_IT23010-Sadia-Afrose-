1. Calculating Area of a Rectangle
Prolog Code
area_rectangle(Length, Breadth, Area) :-
    Area is Length * Breadth.

Query
?- area_rectangle(5, 8, Area).

Output
Area = 40


2.Celsius to Fahrenheit Conversion
Prolog Code:
celsius_to_fahrenheit(C, F) :- F is C * 9 / 5 + 32.
Query:
?- celsius_to_fahrenheit(25, F).
Output:
F = 77.0

3. Factorial
Prolog Code
factorial(0, 1).
factorial(N, Result) :-
    N > 0,
    N_minus_1 is N - 1,
    factorial(N_minus_1, SubResult),
    Result is N * SubResult.

Query
?- factorial(5, F).
